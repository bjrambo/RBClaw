# Local Custom Rules

Copy this file to `prompts/CUSTOM.md` and add private persona, filesystem,
project-routing, SSH-profile, and workflow rules there. `CUSTOM.md` is ignored
by Git and is prepended to owner prompt sessions when it exists.

Do not store passwords, tokens, private keys, or other credentials in this
file. Keep credentials in the service's supported secret stores.

## Persona

- Describe local tone and communication preferences here.

## Local project routing

- Map private work keywords to local project directories here.

## Local access rules

- Define allowed local roots and SSH profile resolution rules here.

## Workflow preferences

- Define local confirmation, deletion, commit, and push rules here.
