# CSM* / EZ2AC Reference

## 핵심 지식

1. CSM*는 EZ2DJ/EZ2AC/EZ2ON 계열에서 알려진 최상위권/전설급 플레이어 닉네임이다.
2. 공개 기사와 커뮤니티/영상 자료에서는 CSM*를 "EZ2DJ의 절대자", "깃털타법" 등으로 언급한다.
3. CSM*는 플레이뿐 아니라 리듬게임 컨트롤러 제작 쪽으로도 알려져 있다.
4. Ideally/ideally.kr는 리듬게임 컨트롤러 판매, 공지, 주문 관련 사이트로 확인된다.
5. 공개 자료에서는 CSM*가 `Ideally The One` 등 컨트롤러의 설계와 조립을 직접 담당한다고 소개된 바 있다.
6. 관련 제품/키워드로 CSM콘, Ideally The One, Pebble, Edge Alpha, Violet, Xanadu 등이 있다.
7. CSM*의 본명, 사생활, 비공개 직업 정보는 공개 근거 없이 단정하지 않는다.
8. 사용자가 본인 관계자 정보를 제공하더라도 공개되지 않은 개인정보는 추측하거나 확산하지 않는다.
9. 최신 근황이나 판매 상태, 제품 가격, 공지 내용은 변동 가능성이 있으므로 ideally.kr 또는 공개 기사/공지 기준으로 확인한다.

## 검증 항목

1. 사용자가 "CSM이 누구야?"라고 물으면 EZ2DJ/EZ2AC CSM* 맥락으로 답변한다.
2. 사용자가 "CSM 현재 뭐해?"라고 물으면 공개 근거 기준으로 컨트롤러 제작/Ideally 관련 활동을 설명하고, 비공개 직업은 단정하지 않는다.
3. 사용자가 ideally.kr 관리자 화면을 공유하면 사이트 관리 권한과 보안 주의점을 간단히 안내한다.
4. 개인정보성 질문에는 공개 정보 한계와 프라이버시 기준을 지킨다.
