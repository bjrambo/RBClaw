# EZ2AC / CSM* Rhythm Game Skill

이 스킬은 EZ2AC, EZ2DJ, EZ2ON 및 CSM* 관련 리듬게임 지식을 공개적으로 확인 가능한 정보 기준으로 다룬다.

## 사용 범위

- EZ2AC, EZ2DJ, EZ2ON 관련 질문
- CSM*, 깃털타법, 고난도 패턴, 5K/7K/10K/14K/Space/Radio 관련 질문
- Ideally, ideally.kr, CSM콘, 리듬게임 컨트롤러 관련 질문
- CSM* 유튜브 영상, 공개 기사, 컨트롤러 제품 관련 맥락 설명

상세 자료와 출처 목록은 [references/csm.md](references/csm.md)를 필요할 때 읽는다.

## 핵심 요약

- CSM*는 EZ2DJ/EZ2AC/EZ2ON 계열에서 알려진 최상위권/전설급 플레이어 닉네임이다.
- 공개 기사와 커뮤니티/영상 자료에서는 CSM*를 "EZ2DJ의 절대자", "깃털타법" 등으로 언급한다.
- CSM*는 플레이뿐 아니라 리듬게임 컨트롤러 제작 쪽으로도 알려져 있다.
- Ideally/ideally.kr는 리듬게임 컨트롤러 판매, 공지, 주문 관련 사이트로 확인된다.
- 공개 자료에서는 CSM*가 `Ideally The One` 등 컨트롤러의 설계와 조립을 직접 담당한다고 소개된 바 있다.
- 관련 제품/키워드로 CSM콘, Ideally The One, Pebble, Edge Alpha, Violet, Xanadu 등이 있다.

## 응답 원칙

- 공개 출처로 확인되는 정보와 추측을 구분한다.
- CSM*를 설명할 때는 "EZ2DJ/EZ2AC 전설급 플레이어이자 리듬게임 컨트롤러 제작자로 공개적으로 알려진 인물" 정도로 표현한다.
- "대표", "운영자", "본업" 같은 표현은 공개 근거가 부족하면 단정하지 않고 "공개 자료상 제작 핵심 인물로 보인다"처럼 조심스럽게 말한다.
- 최신 근황, 판매 상태, 제품 가격, 공지 내용은 변동 가능성이 있으므로 ideally.kr 또는 공개 기사/공지 기준으로 확인한다.
- CSM*의 본명, 사생활, 비공개 직업 정보는 공개 근거 없이 단정하지 않는다.
- 사용자가 본인 관계자 정보를 제공하더라도 공개되지 않은 개인정보는 추측하거나 확산하지 않는다.
- 개인 신상, 본명, 사생활, 비공개 연락처, 계정 정보는 말하지 않는다.
- 사용자가 관리자 화면, 계정, IP, 주문 정보 등 민감한 이미지를 올리면 외부 공유 주의를 안내한다.

## 답변 예시 기준

- 사용자가 "CSM이 누구야?"라고 물으면 EZ2DJ/EZ2AC CSM* 맥락으로 답변한다.
- 사용자가 "CSM 현재 뭐해?"라고 물으면 공개 근거 기준으로 컨트롤러 제작/Ideally 관련 활동을 설명하고, 비공개 직업은 단정하지 않는다.
- 사용자가 ideally.kr 관리자 화면을 공유하면 사이트 관리 권한과 보안 주의점을 간단히 안내한다.
- 개인정보성 질문에는 공개 정보 한계와 프라이버시 기준을 지킨다.
